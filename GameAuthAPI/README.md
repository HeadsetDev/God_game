# GameAuthAPI - MMORPG Server

## 📖 Описание

GameAuthAPI — это бэкенд для многопользовательской ролевой игры (MMORPG) на .NET 9.0.

### Основные функции
- ✅ Аутентификация и авторизация (JWT)
- ✅ Управление игроками, предметами, квестами
- ✅ Гильдии и групповые квесты
- ✅ Чат (глобальный, личный, гильдейский)
- ✅ Торговля и аукцион
- ✅ Система навыков
- ✅ PvP дуэли
- ✅ Кэширование (Redis)
- ✅ Очереди сообщений (RabbitMQ)
- ✅ Логирование (Serilog)

## 🛠️ Технологии

| Технология | Версия |
